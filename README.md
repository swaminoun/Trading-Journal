# Trading Journal Website

A static trading journal website built with HTML, CSS, and JavaScript.

## View Locally

1. Open `index.html` in your browser.
2. Or run a local web server from the project root:
   - Python: `python -m http.server 8000`
   - Node: `npx http-server .`
3. Visit `http://localhost:8000` in your browser.

## Deploy as a Website

This project is ready for deployment with GitHub Pages.

- The deployment workflow is configured in `.github/workflows/deploy-pages.yml`.
- Push to the `main` branch to trigger automatic GitHub Pages deployment.
- If your repository uses a different default branch, update the workflow branch name.

## Structure

- `index.html`: Main website page
- `css/style.css`: Styles and layout
- `js/script.js`: JavaScript file
